# Base-Convert-Encrypt 🔄🔐

A C++ console application built during my 1st semester to explore Discrete Mathematics concepts. This tool handles multi-base number system conversions and basic text encryption/decryption.

## 🚀 Features
- **Number Conversion:** Convert any decimal number into Binary, Octal, or Hexadecimal.
- **Validation:** Includes checks to ensure binary inputs are valid (0s and 1s only).
- **Encryption:** Convert string messages into an array of Decimal, Binary, or Hex values.
- **Decryption:** Revert numeric codes back into the original human-readable string.
- **Terminal UI:** Uses ANSI escape codes and Windows API for a clean, colored console interface.

## 🛠 Tech Stack
- **Language:** C++
- **Header Files:** `<iostream>`, `<Windows.h>`, `<string>`, `<conio.h>`
